# Arsenals-fan

A simple personal website with an Arsenal fan page and a World Cup preview page.

## Files

- `adobe.html` — World Cup 2026 landing page with countdown and host city highlights.
- `intel.html` — Arsenal fan introduction page.
- `My Website.html` — Duplicate fan page with the same content and styling.
- `style.css` — Shared page styles and responsive layout support.

## Notes

- The `intel.html` and `My Website.html` pages now use a shared stylesheet for consistent appearance.
- `adobe.html` has a fixed responsive selector and includes the shared stylesheet for consistent base styling.
